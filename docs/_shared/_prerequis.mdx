:::note Prérequis

Avant de commencer, assurez-vous d'avoir :

- [Node.js](https://nodejs.org/en/download/) **version 18.0 ou supérieure**
  - Lors de l'installation, pensez à cocher toutes les cases liées aux dépendances.
- Un gestionnaire de paquets : `npm` (fourni avec Node.js), `yarn` ou `pnpm`.

:::
